# Mapping_Earthquakes

## Project Overview
The purpose of this project was to create a URL using the Leaflet.js API to populate a geographical map with GeoJSON earthquake data. With the data provided allows the user to view the earthquake data in relation to tectonic plates' location on earth, earthquakes witha magnitude greater that 4.5, and the map shown in dark view.
